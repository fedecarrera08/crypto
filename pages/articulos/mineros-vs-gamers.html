Mineros vs Gamers: lucha por componentes

Es conocido que para minar criptomonedas de manera eficiente se requiere de un buen computador, debido que se necesitan realizar múltiples cálculos de ecuaciones criptográficas, para esto es necesario poseer componentes de alta calidad.

Uno de estos sería la tarjeta gráfica, para poder realizar procesamiento gráfico de los cálculos, también conocidas como unidades de procesamiento gráfico (GPU); estas son las mismas utilizadas por “Gamers” en sus computadores para mejor rendimiento al jugar.

Con el reciente auge de los valores de las criptomonedas a finales del trimestre pasado, este trimestre se ve reflejado un aumento de “Mineros” para entrar a este nuevo mercado, por ende, ocurre un aumento de demanda de componentes de computadoras.

Se puede observar que el desarrollo de las criptomonedas esta afectando el mercado de hardware, hay que esperar los reportes fiscales de las demás compañías de diversos componentes, como Intel con respecto a procesadores, para observar mejor la tendencia.
